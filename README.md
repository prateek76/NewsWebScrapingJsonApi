# NewsWebScrapingJsonApi
Get JSON data of news to use in apps and websites  

## Built With

* [Python](#) - Script
* Urllib
* [beautiful soup](#) - web scrapping

## Development setup

Install following your OS (Windows or linux)

```sh
urllib
```

```sh
beautiful soup
```

## usage

Windows & Linux:

Run the script

```sh
python3 script.py
```
